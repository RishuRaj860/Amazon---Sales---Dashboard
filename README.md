# Amazon Sales Dashboard

## Tools Used
- Excel
- Pivot Tables
- Pivot Charts

## Dashboard Includes
- Category Wise Sales
- Date Wise Sales
- Month Wise Sales
- State Wise Sales
- Order Status Analysis

## Key Insights
- T-shirt is the highest selling category.
- April generated the highest sales.
- Most orders were shipped successfully.
